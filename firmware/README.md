# FIRMWARE UPDATE

To activate Wi-Fi, press the module button for more than 5 seconds.<br>
After releasing the button, the LED must blink in blue.<br>
The blue LED indicates that Wi-Fi is enabled on the module.

Connect to the module's captive Wi-Fi :<br>
&nbsp;&nbsp;&nbsp;&nbsp;SSID = APW3<br>
&nbsp;&nbsp;&nbsp;&nbsp;Password = ToTheMoon

Open in your browser :<br>
&nbsp;&nbsp;&nbsp;&nbsp;http://192.168.4.1/

<img src="/images/APW3 Module - Menu MAIN.png">

Navigate to the "Update" menu.

<img src="/images/APW3 Module - Menu Firmware Update.png">

Select the firmware_XXXX.X.X.bin file on your computer and click UPDATE.

<img src="/images/APW3 Module - Menu Firmware Successful.png">

After a few seconds, the module should automatically restart with the new firmware installed.<br>
The LED should then be green.

If the LED is still blinking in blue, the firmware has not been installed.<br>
In this case, please try again.

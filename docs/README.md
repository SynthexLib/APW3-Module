# LED 
A green LED indicates that the module is powered on and ready to operate.

A flashing green LED indicates that the module is communicating with the control board.

A blinking blue LED indicates that Wi-Fi is enabled on the module.

A blinking red LED indicates a reset to factory settings.

# BUTTON
A short press of the button allows you to change the brightness of the LED.

Pressing the button for 5 seconds will activate or deactivate the module's Wi-Fi connection. 

Pressing the button for 40 seconds erases all settings and returns to the factory configuration.

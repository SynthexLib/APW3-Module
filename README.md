# APW3 Module
This module allows you to control the voltage of an APW3 power supply <br>
from 12v to 14v, directly from an Antminer S19/S21 control board !

This module emulates the entire communication protocol <br>
to make it appear as if it were a genuine S19/S21 PSU.

- It uses a standard 4-pin Antminer voltage regulation cable
- A single button to adjust the LED brightness and access the configuration menu
- The RGB LED allows you to monitor activity and configuration
- Has its own web server for configuring the module
- Wireless Firmware Update by direct captive Wi-Fi
- Select the PSU to simulate from all available options
